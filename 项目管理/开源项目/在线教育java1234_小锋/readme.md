**本项目是专门为像我这种出身卑微的社会底层穷二代技术狗创业用的，所以烧钱的功能全部去掉，尽可能的省钱；**

本项目设计要求用户界面简约大气，用户操作上简单易用，以及去掉烧钱，维护复杂的功能；

基于以上设计要求，笔者阉割和简化了如下需求功能：

1，登录功能仅支持QQ第三方登录和微信扫码登录，不支持短信和其他账户注册登录，防止被恶心攻击，浪费创业者的精力。

2，支付功能仅支持支付宝和微信；

3，不支持课程评论功能；

4，不支持在线播放课程功能，流量烧钱，维护成本也高，免费课程直接链接b站，收费课程直接加密离线学习；

5，大部分功能是微信群，QQ群维护粉丝和会员，线上维护代价大，成本高。
